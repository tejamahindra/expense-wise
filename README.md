
# ExpenseWise 💰
A full-stack personal finance tracker to manage income, expenses and view monthly analytics.

## Tech Stack
- **Backend:** Java 17, Spring Boot 3.2, Spring Security (JWT), REST APIs
- **Frontend:** Angular 17, Chart.js, Bootstrap 5
- **Database:** MySQL (AWS RDS)
- **DevOps:** Docker, Jenkins CI/CD, AWS EC2 + S3 + RDS

## Features
- User registration & login with JWT authentication
- Add, edit, delete income & expenses by category
- Monthly analytics dashboard with pie & bar charts
- AI-powered spending insights
